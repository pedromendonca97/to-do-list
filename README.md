# To-Do List App (Node.js)
Aplicativo de gerenciamento de tarefas (To-Do List) construído com Node.js. Este projeto é voltado para estudo de fundamentos de backend, manipulação de dados e rotas HTTP.

# Tecnologias e Bibliotecas
- Node.js – ambiente de execução JavaScript.
- HTTP (módulo nativo) – para criar o servidor.
- UUID – gerar IDs únicos para tarefas.

# Funcionalidades
- Criar tarefa (POST /tasks)
- Listar tarefas (GET /tasks)
- Procurar tarefa por ID (GET /tasks/:id)
- Atualizar tarefa (PUT /tasks/:id)
- Deletar tarefa (DELETE /tasks/:id)

# Como rodar o projeto
- Clone o repositório
  - git clone https://github.com/pedromendonca97/to-do-list.git
  
- Entre na pasta do projeto
  - cd to-do-list
 
- Instale dependências
  - npm install
 
- Inicie o servidor
  - node server.js
